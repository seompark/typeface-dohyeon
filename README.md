# typeface-dohyeon

## Install
```
npm install --save typeface-dohyeon
```

## How to apply?

### On module bundler(webpack)
```
import 'typeface-dohyeon'
```
or
```
require('typeface-dohyeon')
```

then..

```
font-family: 'BM DoHyeon';
```
